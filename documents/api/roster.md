# Endpoints for Player

prefix: `api/v1/`

-   **POST** `players/`
    Create a new player

-   **GET** `players/`
    Get all players

-   **GET** `players/{id}/`
    Get the specific player

-   **PATCH** `players/{id}/`
    Change the specific player information

-   **DELETE** `players/{id}/`
    Delete the specific player
